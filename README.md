## Features

### Part Measurement
- Part detection from camera input  
- Dimensional measurement (length, width, diameter, etc.)  
- Measurement data output to PLC or robot controller  


### Part Inspection
- Vision-based quality inspection  
- Binary result: **OK / NOT OK**  
- Inspection result output to PLC or robot controller  


### Code Reader (Barcode / QR)
- Barcode and QR code detection  
- Code data extraction  
- Decoded value output to PLC or robot controller  


### Robot Picker *(Discontinued)*
- Part detection in robot workspace  
- Pick-point coordinate calculation  
- Coordinate transfer to robot controller  

**Status:** Incomplete and not production-ready
